## Problem

**Headline**
```
You already know this part
```

**Body**
```
Documenting a workflow means print screen, paste, scroll, print screen again,
then lining the pieces up in a doc so a client can follow it. The full-page
screenshot tools don't help. The builder never scrolls, so they only ever grab
the part you can see.

So most workflows never get documented. Then a client asks what happens after
someone books, and you're back in the builder taking pictures of it again.
```

---

## Solution

**Headline**
```
One image. Every step.
```

**Body**
```
Open a workflow, click the icon, click Capture. A few seconds later there's a
PNG in your Downloads named after the workflow, with every step in it and the
text still readable.

It resets the canvas to 100% before it starts, so it doesn't matter how zoomed
out you were. The minimap and zoom buttons are left out, so the image is clean
enough to drop straight into an SOP.
```

---

## How it works (3 steps)

```
1. Open a workflow
   Any workflow in the GoHighLevel builder, including whitelabel accounts.

2. Click Capture
   The extension pans across the workflow and takes the shots for you.

3. Get one image
   PNG or JPEG, named after the workflow, straight to your Downloads.
```

## Privacy

**Headline**
```
It doesn't send anything anywhere
```

**Body**
```
The extension makes no network requests at all. Nothing is collected, nothing is
transmitted, nothing is sold. There's no account and no analytics. The capture
happens in your browser and the image goes to your Downloads.

It only runs on the workflow builder page. Every other tab is untouched.

The code is public if you want to check: github.com/Beeyach/workflow-screenshotter
```

## FAQ

```
Is it really free?
Yes. No paid tier, no trial, no upsell.

Does it work on my whitelabel domain?
Yes. The builder is the same underneath, so custom domains work.

Does it work with Zapier, Make, or n8n?
No. GoHighLevel workflows only.

What about very long workflows?
Those are the ones it's for. Long workflows take longer because it captures in
sections, and you'll see the progress while it runs.

Can I get a bigger image for printing?
Yes. There's a size setting in the popup. Normal matches what you see on screen.
2x and 3x are there if you need to zoom in or print.

Who made this?
We did. We build booking and follow-up systems in GoHighLevel, and we got tired
of screenshotting workflows by hand.
```
